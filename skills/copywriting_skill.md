# Copywriting Expert Skill
## Charming by Vik — Brand Voice Guide

> Activate this skill whenever writing any content for the brand:
> website copy, product descriptions, workshop texts, CTAs, social captions, or any marketing material.
> This skill ensures every word reflects the soul of Charming by Vik.

---

## 1. Brand Personality

Charming by Vik is not just a jewelry brand — it is a feeling.
Every piece of content should make the reader feel **seen, inspired, and invited**.

| Attribute       | Description                                                                 |
|-----------------|-----------------------------------------------------------------------------|
| Warm            | Speak to the reader like a trusted friend who happens to be an expert       |
| Elegant         | Refined language — no slang, no noise, only words that add value            |
| Personal        | Use "את" and "שלך" — speak directly to one woman, not a crowd               |
| Artistic        | Celebrate the craft, the materials, and the hands that made the piece       |
| Empowering      | Jewelry is not decoration — it is self-expression and confidence            |

---

## 2. Tone of Voice

### Always sound like this:
> "כל צ'ארם שנבחר הוא רגע שנשמר — זיכרון שתוכלי לשאת קרוב ללב."

### Never sound like this:
> "מוצר איכותי במחיר משתלם! קני עכשיו!"

### The three pillars of the tone:
1. **קסם** — there is magic in handmade things; name it, evoke it.
2. **ייחודיות** — every piece is one of a kind, just like the woman wearing it.
3. **רגעים בלתי נשכחים** — jewelry marks life's milestones; write with that weight.

---

## 3. Language Rules

- **Script**: High-quality Modern Hebrew, fully RTL-formatted.
- **Register**: Elevated but accessible — no slang (`חיים`, not `חייםל`), no overly academic terms.
- **Pronouns**: Address the reader directly as **את** (singular feminine).
- **Sentence length**: Short to medium. One idea per sentence. Let sentences breathe.
- **Paragraph length**: 2–4 lines maximum. White space is part of the design.
- **Punctuation**: Use the em dash (—) for elegant pauses. Avoid excessive exclamation marks (!).
- **Brand name**: Always written as "Charming by Vik" in Latin — never translated.

### Power Words to use freely:
`קסם` · `עדין` · `ייחודי` · `מיוחד` · `בעבודת יד` · `אישי` · `רגע` · `זיכרון`
`אהבה` · `נוצץ` · `ברק` · `אלגנטי` · `נשי` · `יצירה` · `חוויה` · `השראה`

### Words to avoid:
`זול` · `מבצע` · `קני` · `מהר` · `מוצר` · `פריט` (use `תכשיט` instead) · `סחורה`

## 4. Content Structure

Every piece of content — regardless of length — should follow this arc:

```
1. Hook       → Evoke an emotion or paint a picture in one sentence
2. Body       → Describe the value: materials, craft, feeling, occasion
3. CTA        → A gentle, inviting call to action (never a demand)
```

### Use this formatting for readability:
- **Short intro paragraph** (2–3 lines) to set the scene.
- **Bullet points** for features, materials, or workshop details — clean and scannable.
- **CTA** on its own line, set apart visually.

## 5. Materials & Craft Vocabulary

When describing jewelry, use precise, evocative language about materials:

| Material         | How to describe it                                      |
|------------------|---------------------------------------------------------|
| זהב ורוד (Rose Gold) | "ציפוי זהב ורוד עדין שמחמיא לכל גוון עור"           |
| כסף (Silver)     | "כסף מוברק בגימור עדין ומינימליסטי"                   |
| צ'ארמים (Charms) | "צ'ארמים קטנים שכל אחד נושא משמעות משלו"              |
| חרוזים (Beads)   | "חרוזים בצבעים עדינים שנבחרו ביד לתיאום מושלם"        |
| חוט (Wire/Thread)| "חוט עדין ועמיד המחזיק את היצירה בשלמותה"             |

Always emphasize:
- That items are **בעבודת יד** (handmade).
- That materials are chosen **בקפידה** (with care).
- That each piece is **ייחודי** — no two are exactly the same.

## 6. Call to Action (CTA) Rules

CTAs must feel like **an invitation**, not a command.

### Approved CTA Formulas:

| Context               | CTA                                              |
|-----------------------|--------------------------------------------------|
| Shop / collection     | "גלי את הקולקציה שמחכה לך"                      |
| New collection        | "לקולקציה החדשה"                                 |
| Workshops             | "בואי ליצור איתנו"                               |
| Workshop signup       | "הרשמי לסדנה הקרובה"                             |
| Gift / occasion       | "מצאי את המתנה המושלמת"                          |
| General engagement    | "בואי נספר את הסיפור שלך"                        |
| Learn more            | "גלי עוד"                                        |

### Never use:
- "קני עכשיו" — too aggressive
- "אל תפספסי!" — creates anxiety, not desire
- "מבצע לזמן מוגבל" — cheapens the brand

## 7. Content Templates

### Product Description Template
```
[Hook — one evocative sentence about the feeling or occasion]

[Body — 2–3 lines describing the piece: materials, craft, personalization options]

• [Feature bullet 1]
• [Feature bullet 2]
• [Feature bullet 3]

[CTA — inviting, one line]
```

**Example:**
> צמיד שמספר את הסיפור שלך, צ'ארם אחרי צ'ארם.
>
> כל צמיד מורכב ביד מחוט עדין וצ'ארמים שנבחרו בקפידה — זמין בציפוי זהב ורוד, כסף, וזהב צהוב.
>
> • התאמה אישית מלאה — בחרי את הצ'ארמים שמדברים אלייך
> • זמין בשלושה גוונים מתכתיים
> • מתנה מושלמת לכל אירוע
>
> גלי את הקולקציה שמחכה לך →

---

### Workshop Description Template
```
[Hook — describe the experience, not the product]

[Body — what participants will do, learn, and feel]

• [Detail: duration / group size / price]
• [Detail: skill level required]
• [Detail: what they take home]

[CTA — warm and personal]
```

**Example:**
> ערב אחד. קבוצה קטנה. תכשיט שיישמר לנצח.
>
> בסדנת צמידי הצ'ארם תלמדי את יסודות עבודת התכשיטים — בחירת חומרים, שילוב צ'ארמים, ובניית לוק שלם. הכל בהנחיה אישית של ויק, באווירה חמה ומעוררת השראה.
>
> • משך: שעתיים | עד 8 משתתפות
> • מתאים לכל רמת ניסיון
> • יוצאות עם צמיד מוגמר
>
> בואי ליצור איתנו →

## 8. Skill Activation Checklist

Before publishing any content, verify:

- [ ] Written in elegant, high-quality Hebrew
- [ ] Addresses the reader as **את** (singular feminine)
- [ ] Contains at least one power word from the approved list
- [ ] Paragraphs are short (max 4 lines)
- [ ] Includes a CTA that invites — not demands
- [ ] Materials described with precision and warmth
- [ ] No aggressive sales language
- [ ] Consistent with the brand name "Charming by Vik"
